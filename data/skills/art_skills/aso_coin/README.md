# ASO-coin · 金币识别 App 商店截图风格

## 定位

面向 App Store / Google Play 的 **9:16 竖屏 ASO 商店截图**，用于金币识别与估值类 App。

## 核心视觉

- **背景**：哑光黑 / 炭灰渐变，微弱金色环境光
- **主体**：写实金币微距摄影（金属浮雕清晰、边缘高光）
- **设备**：iPhone 竖屏 mockup，UI 简洁现代
- **构图**：单屏或 2–3 屏横展全景 ASO（背景与金币可跨屏延续）
- **文字**：英文大标题 + 说明文案，金色或白色，高可读

## 适用场景

商店截图、Feature Graphic、社媒投放、ASO A/B 测试变体

## 严禁

卡通币、塑料感、霓虹荧光、杂乱背景、水印、竞品 Logo、虚假收益承诺
